# HireVision: The Next-Gen AI Recruiter

Welcome to the **HireVision** repository, a fully functional AI recruitment platform rebuilt for the **Gemini 3 Hackathon**.

## Overview

HireVision leverages the power of **Google Gemini 3.0 Flash** to provide distinct advantages:
- **Ultra-Fast Video Analysis**: Analyze candidate interviews in real-time.
- **Multimodal Resume Reasoning**: Understand visual resumes beyond text.
- **Intelligent Culture Matching**: Go deeper than keywords to assess fit.

## Quick Start
1.  Clone this repo.
2.  Navigate to `hirevision` (the core app).
3.  `npm install --legacy-peer-deps`
4.  Set `GOOGLE_GENAI_API_KEY` in `.env.local`.
5.  `npm run dev`

## Updates for Gemini 3 Hackathon
- **Model**: Upgraded to `gemini-3.0-flash`.
