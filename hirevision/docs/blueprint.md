# **App Name**: HireVision

## Core Features:

- AI Resume Parsing: Automatically parse resume data using SmartInference to extract key information and populate candidate profiles, allowing users to store resume contents with SmartBuckets.
- Candidate Scoring: Score candidates based on job requirements using SmartInference, providing a fit score to rank applicants effectively.
- Automated Email Generation: Generate personalized emails for candidate communication (invites, rejections) using SmartInference to save time and maintain consistency.
- Job Posting Management: Create, edit, and manage job postings with detailed descriptions, required skills, and application workflows.
- Candidate Application Tracking: Track candidates through the application pipeline with status updates, interview scheduling, and feedback collection. Update SmartSQL database accordingly.
- Role-Based Access Control: Implement RBAC to ensure data security and appropriate access levels for different user roles (Owner, Recruiter, Hiring Manager, Interviewer, Guest).
- Dashboard Analytics: Visualize key recruiting metrics (total jobs, active applications, time-to-fill) with interactive charts and AI-generated insights tool, pulled from SmartSQL.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a professional and trustworthy feel, reflecting competence and innovation.
- Background color: Light gray (#F5F5FA), a desaturated indigo, providing a clean and modern backdrop for content.
- Accent color: Teal (#4CAF50), an analogous hue, to highlight key interactive elements such as call-to-action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look that works well for both headlines and body text. 
- Code font: 'Source Code Pro' for displaying code snippets.
- Consistent, clean icons that visually represent job functions and data metrics, making navigation intuitive.
- SaaS-style layout with a clear left sidebar (desktop) and bottom navigation (mobile), optimizing screen real estate.
